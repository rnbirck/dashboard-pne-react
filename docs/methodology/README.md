# Metodologias

Estes documentos descrevem o comportamento implementado. Código, testes e dados gerados continuam sendo a evidência executável; divergências ou decisões humanas pendentes ficam em [`data-gaps.md`](data-gaps.md), nunca são resolvidas por inferência documental.

| Documento | Leia quando |
| --- | --- |
| [`eja-articulada.md`](eja-articulada.md) | Alterar EJA articulada, denominador da EJA, metas 10/12.c ou detalhes relacionados. |
| [`medio-tecnico-articulado.md`](medio-tecnico-articulado.md) | Alterar ensino médio técnico integrado, Meta 12.a ou dados concomitantes. |
| [`pne2026-trend.md`](pne2026-trend.md) | Alterar tendência, elegibilidade, limiar ou apresentação direcional. |
| [`state-reference-pne2014.md`](state-reference-pne2014.md) | Alterar a referência RS do ciclo encerrado. |
| [`state-reference-pne2026.md`](state-reference-pne2026.md) | Alterar a referência RS do ciclo vigente, infraestrutura ou projeções estaduais. |
| [`data-gaps.md`](data-gaps.md) | A fonte não oferece insumo seguro ou há divergência a validar. |

Ao mudar método, mantenha chave, definição, fórmula, fonte, período, granularidade, política de nulos, consumidores, testes e limitações sincronizados. Não regenere JSON nem altere fórmula como efeito colateral de uma edição documental.
